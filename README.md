# steal-css-style-tag

[StealJs](https://github.com/stealjs/steal) plugin for loading `css` using `style` tag instead of `link`.

## usage

Configure in `package.json`
```js
"system": {
  "map": {
    "$css": "steal-css-style-tag",
  },  
}
```
## install
```bash
npm install steal-css-style-tag
```
